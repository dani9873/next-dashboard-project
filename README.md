# Next.js Dashboard

A modern dashboard application built with **Next.js**, following best practices such as the App Router, server components, and TypeScript.

This project is based on the official Next.js learning course and structured to reflect real-world development workflows.

---

## 🚀 Features

* ⚡ Built with Next.js App Router
* 🧩 Component-based architecture
* 📦 TypeScript for type safety
* 🗂 Organized project structure (`app`, `lib`, `ui`)
* 🧪 Placeholder data for development
* 🎨 Pre-styled UI components

---

## 📁 Project Structure

```
/app        → Routes, pages, and layouts  
/app/lib    → Utilities, data fetching, types  
/app/ui     → UI components  
/public     → Static assets  
```

---

## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/dani9873/nextjs-dashboard.git
cd nextjs-dashboard
```

### 2. Install dependencies

```bash
pnpm install
```

### 3. Run the development server

```bash
pnpm dev
```

Now open http://localhost:3000 in your browser.
